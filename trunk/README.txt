--- About
-------------------------------------------------------------------------------

The MokoIAX project is an effort to integrate VoIP functionality into the
OpenMoko platform.  Specifically, this effort is to use the IAX protocol.

IAX stands for Inter Asterisk eXchange, and was originally developed for use
with the Asterisk telephony platform (www.asterisk.org).  However, it has since
been adopted by other open source platforms, as well.

IAX is a great protocol for use on a mobile device for a number of reasons.
The most notable reason is that it does not require any extra effort to work
in different types of networks, especially those involving NAT.  With a mobile
device where you roam amongst various networks that one does not control, using
a protocol that "just works" will prove extremely beneficial.
